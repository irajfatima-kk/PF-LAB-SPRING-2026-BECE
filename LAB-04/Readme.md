# Lab 04 - Programming Fundamentals

This lab contains tasks using if-else, switch statements, and logical operators.

## Tasks
1. **Sports Team Selection** – Check if attendance ≥ 75% for eligibility.
2. **Internet Data Usage** – Categorize users as Basic, Standard, or Heavy.
3. **Bank Account Balance** – Show Positive, Overdrawn, or Zero balance.
4. **Wi-Fi Authentication** – Verify username and password.
5. **Banking Menu** – Use switch for Balance Inquiry, Withdraw, Deposit, Exit.
6. **Scholarship Evaluation** – Calculate percentage and decide scholarship type.
7. **Festival Sale Discount** – Apply discount based on purchase amount.
8. **Math Operations Console** – Perform addition, subtraction, multiplication, division, square, cube, square root, power, absolute value, modulus.
