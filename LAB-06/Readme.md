# Programming Fundamentals - Lab 06 (C Language)

This repository contains solutions to **Lab 06 tasks** for Programming Fundamentals.  
Each task demonstrates the use of **loops in C** (for, while, do-while) with practical examples.

##  Tasks Included
1. Student Attendance Counter (for loop)
2. ATM Withdrawal System (while loop)
3. Password Retry System (do-while loop)
4. Sum of First N Natural Numbers (for loop)
5. Temperature Monitoring System (while loop with sentinel)
6. Multiplication Table Generator (for loop)
7. Menu-Driven Calculator (do-while loop)
8. Number Guessing Game (while loop)
9. Factory Production Planning System (for loop)
10. University Data Analysis System (while loop with classification)
11. Online Shopping Cart Checkout System (do-while loop with discount logic)
12. Bus Seat Reservation Monitoring System (while loop with condition control)
